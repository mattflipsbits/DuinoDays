## Day 1: Explaining Blink

### Today's Goals
- Learn the basic anatomy of an Arduino board
- Learn the basic structure of an Arduino program
- Configure an Arduino pin as either input or ouput using the `pinmode()` function
- Control an Arduino pin using the `digitalWrite()` function
- Pause the Arduino board using the `delay()` function

### Learn
1. [Getting Started: Anatomy of an Arduino Board](https://docs.arduino.cc/learn/starting-guide/getting-started-arduino/#anatomy-of-an-arduino-board)
2. [Arduino Sketches](https://docs.arduino.cc/learn/programming/sketches/)
3. [pinMode()](https://docs.arduino.cc/language-reference/en/functions/digital-io/pinMode/)
4. [digitalWrite()](https://docs.arduino.cc/language-reference/en/functions/digital-io/digitalwrite/)
5. [delay()](https://docs.arduino.cc/language-reference/en/functions/time/delay/)

### Challenge
- Modify the Blink example to blink [SOS in Morse Code](https://en.wikipedia.org/wiki/SOS)
