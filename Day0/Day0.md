## Day 0: Setting Up Shop
The goal for today is to set up and familiarize yourself with Tinkercad.

## From the Docs
- [What is Arduino?](https://docs.arduino.cc/learn/starting-guide/whats-arduino/)

### Activities
- [Create a Tinkercad Account](https://www.tinkercad.com/join)
- Join the ['Duino Days Tinkercad Class](https://www.tinkercad.com/joinclass/KB9BWQB6A)
- In the Tinkercad Class, open ***Day 0*** then Copy and Tinker the ***Basic Tinkercad*** activity
- Using the notes as a guide, make sure you can move and zoom the work area,
  use the Rotate and Mirror Tools, connect components using the wire tool,
  open and close the Code window, and Start and Stop the simulation.
